## Getting Started

```sh
bun run something
bun install
bun wiptest
```

```sh
bun add something
bun remove something
```

## Using bun with Next.js

```sh
bun create next ./next
cd next
bun dev
```

## Using bun with CRA

```sh
bun create react ./cra
cd cra
bun dev
```
